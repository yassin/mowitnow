# mowitnow

Summary
-------
program of Mower implementation.


Dependencies
------------
* JDK 1.7
* Maven 3
* Log4j
* TestNG


Compile & Run
-------------

    mvn clean install
    cd target
    java -jar mowitnow-<version>.jar  < absolute file path >
    

Javadoc
---------------------
Java Doc is generated in target/apidocs 
you can open index.html in the browser to see all javaDoc.
